# Walmart-Sales-Dashboard

## About This Project

Hi! My name is Reynard, and this is my first Microsoft Excel portfolio project.

In this project, I cleaned, analyzed, and visualized Walmart sales data using Microsoft Excel. The dataset contains weekly sales information from multiple Walmart stores across the United States between 2010 and 2012.

## Dataset

The dataset was created by **Mikhail** and published on Kaggle.

Source:
https://www.kaggle.com/datasets/mikhail1681/walmart-sales/data?select=Walmart_Sales.csv

Special thanks to the original author for making this dataset publicly available.
